#Task Manager

## Description
A full stack Task Manager application with register,login and adds and manages tasks securely.

---

## Features
- User registration and login(JWT)
- Password hashing using bycrypt
- Create Tasks
- View Tasks
- Update Tasks
- Delete Tasks

---

## Tech Stack
- Frontend:React.js
- Backend: Node.js,Express.js
- Database:MongoDB

---

## Backend Setup

```bash
cd backend
npm install
```

---

## Backend Run
```bash
npm run dev
```

---

## Frontend Setup

```bash
cd src
npm install
```

---
## Frontend Run
```bash
npm start
```
